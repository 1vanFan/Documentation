# 发布和订阅音频流
在发布或订阅音频流前，请确保你已完成环境准备、安装包获取等步骤，并成功加入频道，详见[客户端集成](../../cn/Voice/windows_video.md)。

## 实现方法
Agora SDK for Windows 默认开启语音功能。用户加入频道后，无需调用其它方法，可以直接发布或订阅音频流。
