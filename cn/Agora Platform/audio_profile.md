# 音频编码属性
音频编码属性是指本地编码音频的采样率、编码策略、声道数、码率等属性。

声网提供 API 给开发者设置理想网络状态下的音频编码属性，详见[设置音频编码属性](https://docs.agora.io/cn/Interactive%20Broadcast/audio_profile_android)本地音频流在编码后即会发送给频道内远端用户，故开发者对音频编码属性的设置会影响远端用户接收的音频数据。
