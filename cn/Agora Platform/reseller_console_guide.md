# 代理商控制台概览
欢迎使用 Agora 代理商控制台（Agora Reseller Console）。代理商控制台是 Agora 为代理商准备的公司管理工具，通过直观的界面，方便代理商对使用 Agora 产品的客户公司进行用量查看、状态管理等操作。本指南简要介绍如何使用代理商控制台。

## 注册与登录

### 创建新账号

使用代理商控制台前，你需要联系 sales@agora.io 或在 Agora Dashboard [提交工单](https://dashboard.agora.io/support)申请一个代理商控制台账号。请务必保存好你的账号和密码，若不小心遗失密码，参照[找回密码](#ForgetPW)中的步骤即可轻松找回密码。

成功创建账号后，你可以登录[代理商控制台](https://reseller.agora.io/)，创建公司并查看其用量等。

### 修改账户设置和密码

登录[代理商控制台](https://reseller.agora.io/)，点击顶部导航栏右侧账号名，在下拉菜单中点击**设置**，进入账号设置页面。

![](https://web-cdn.agora.io/docs-files/1569553517030)

- 在**用户信息**栏，你可以查看用户信息、更改界面语言和重置密码。
  ![](https://web-cdn.agora.io/docs-files/1569571906293)
- 在 **API 密钥**栏，你可以生成用于 RESTful API 签名的 **Access Key ID** 和 **Access Key Secret**。详见 [RESTful API 签名](https://docs.agora.io/cn/Agora%20Platform/reseller_console_restful?platform=All%20Platforms#签名)。
   Access Key ID 是用于标识访问者身份的 ID，Access Key Secret 是用于加密签名字符串和服务器端验证签名字符串的密钥，必须严格保密。
  ![](https://web-cdn.agora.io/docs-files/1569571934288)
  输入使用密钥的目的，并点击生成密钥，你可以在输入框下方看到密钥。一个代理商可配多个 Access Key，同一时间可有多个激活的 Access Key。

  <div class="alert note">为加强保密性，页面刷新后，你将无法看到该密钥。请及时复制保存。</div>

  对于已生成的密钥，你还可以点击**删除**进行删除。删除后，该密钥记录会从列表中消失，且应用该密钥进行认证的 RESTful API 会无法使用。

<a name="ForgetPW"></a>
### 找回密码

如果你忘记了密码，可通过以下方式找回密码：

1. 进入[登录页面](https://reseller.agora.io/)。
2. 点击**忘记密码**。
   ![](https://web-cdn.agora.io/docs-files/1569571994260)
3. 填写注册时使用的邮箱地址。Agora 会向你的邮箱发送一封密码重置邮件。
4. 登录你的邮箱，根据邮件中的指引重置密码。

## 创建和管理公司

### 创建公司

创建公司的步骤如下：

1. 登录[代理商控制台](https://reseller.agora.io/)，在左侧导航栏中点击**账号>公司，**找到**注册链接**，复制并打开该链接。通过该链接创建的公司会自动归属于该代理商账号。
   ![](https://web-cdn.agora.io/docs-files/1569571303367)
2. 根据界面提示创建公司，即[创建新 Dashboard 账号](https://docs.agora.io/cn/Agora%20Platform/sign_in_and_sign_up?platform=All%20Platforms)。若你需创建的公司已有 Dashboard 账号，联系 sales@agora.io 创建并提供该公司名称。
3. 成功创建公司后，你可以对该公司进行管理。

### 管理公司

对于已创建的公司，你可以在**公司列表**中进行管理。

![](https://web-cdn.agora.io/docs-files/1569572009737)

#### 搜索

1. 在**公司列表**右上角选择搜索类型： CID（公司账号的唯一 ID）、用户名或邮箱。

2. 输入对应的 CID、用户名或邮箱后，点击**搜索**。

 <div class="alert note">当搜索类型为 CID 时，不支持模糊搜索，需输入完整 CID。</div>

3. 目标对象会在列表中显示。

#### 查看公司状态

在**公司列表**状态栏中，你可以查看公司的状态。

- 正常：表示公司状态正常，可正常使用 Agora 产品。
- 手动停用：表示公司已被冻结，该公司无法继续使用 Agora 产品。

#### 冻结或移除公司

在**公司列表**中点击**详情**，你可以查看详细的公司及用户信息，并进行以下操作：

![](https://web-cdn.agora.io/docs-files/1569572021969)

- 冻结公司。点击**冻结账户**，在弹出窗口中选择冻结账户原因及是否禁止 Dashboard 登录，并点击**冻结账户**。
  - 成功冻结后，该公司财务状态会变为**手动停用**，即无法继续使用 Agora 产品。同时，系统也会发送**账户停用通知**邮件给该用户。
  - 若同时勾选**禁止 Dashboard 登录**，则该公司账号状态也会变为**禁用**，即该公司无法访问 Dashboard。

- 将该公司从你账户中移除。点击**将该公司从你账户中移除**并确认后，该公司会从**公司列表**中消失，你将无法对该公司进行操作。

- 点击**以该公司访问 Dashboard**，可以进入该公司的 Dashboard 界面。

#### 查看用量

在**公司列表**中点击**用量**，可以快速查看该公司的用量。详情见[查看用量](#CheckUsage)。

## 查看和下载用量

<a name="CheckUsage"></a>
### 查看用量

登录[代理商控制台](https://reseller.agora.io/)，在左侧导航栏中点击**用量>图表**后，你可以进行以下操作：

![](https://web-cdn.agora.io/docs-files/1569572036816)

- 选择查询时段及图表显示间隔（每日或每月）。
- 输入公司名查询目标公司的用量。若不输入公司名，则用量图表中显示的是所有公司的用量总和。
- 查看不同产品的用量，包含音视频、本地服务端录制、云端录制等。你可以移动鼠标至图表查看某日/某月的用量，或在页面下方表格中查看用量详情。

#### 音视频

选择**音视频**可查看音视频的用量。其中，每日用量为目标公司所有项目当天用量的累计值，每月用量为每日用量的累计值。

![](https://web-cdn.agora.io/docs-files/1569572048665)

- 单位：分钟。
- 图表内容：默认显示主播和观众音频时长及视频总时长。若需查看主播和观众 SD 视频总时长、HD 视频总时长和 HDP 视频总时长图表，可在图表横坐标下方图例中选择。

 <div class="alert note">主播和观众视频总时长为主播和观众 SD 视频总时长、HD 视频总时长及 HDP 视频总时长之和。
	<br>- SD：标清，视频分辨率的取值范围为 (16p × 16p,640p × 360p) 。
	<br>- HD：高清，视频分辨率的取值范围为 [640p × 360p,1280p × 720p]。
	<br>- HDP：超高清，视频分辨率大于 1280p × 720p。</div>

#### 本地服务端录制

选择**本地服务端录制**可查看本地服务端录制的用量。其中，每日用量为目标公司所有项目当天用量的累计值，每月用量为每日用量的累计值。

![](https://web-cdn.agora.io/docs-files/1569572060889)

- 单位：分钟。
- 图表内容：默认显示观众音频时长、观众 SD 视频时长、观众 HD 视频时长和观众 HDP 视频时长。若需隐藏图表，可在图表横坐标下方图例中取消选择。

#### 云端录制

选择**云端录制**可查看云端录制的用量。其中，每日用量为目标公司所有项目当天用量的累计值，每月用量为每日用量的累计值。

![](https://web-cdn.agora.io/docs-files/1569572070370)

- 单位：分钟。
- 图表内容：默认显示观众音频时长、观众 SD 视频时长、观众 HD 视频时长和观众 HDP 视频时长。若需隐藏图表，可在图表横坐标下方图例中取消选择。

#### 小程序

选择**小程序**可查看小程序的用量。其中，每日用量为目标公司所有项目当天用量的累计值，每月用量为每日用量的累计值。

![](https://web-cdn.agora.io/docs-files/1569572082196)

- 单位：分钟。
- 图表内容：默认显示主播和观众音频时长及视频总时长。若需隐藏图表，可在图表横坐标下方图例中取消选择。

#### 转码

选择**转码**可查看转码的用量。其中：

- 按每日查询时，用量数据为该公司当天每个项目用量峰值之和。
- 按每月查询时，用量数据为该公司当月内每个项目用量峰值之和，而不是每日用量数据之和。

![](https://web-cdn.agora.io/docs-files/1569572092111)

- 单位：分钟。
- 图表内容：默认显示总音频连麦时长、H264 音频连麦时长、H264 单主播连麦时长、H264 SD 连麦视频时长、H264 HD 连麦视频时长及 H264 HDP 连麦视频时长。若需查看 H265 相关图表，可在图表横坐标下方图例中选择。
  - H264 音频：主播的音频总时长。
  - H264 单主播：使用 H.264 编码的单主播视频时长。
  - H265 音频连麦：连麦时，主播的音频总时长。

<div class="alert note">关键术语解释：
	<br>- 转码：在推流到 CDN 过程中，发送到 SD-RTN™ 的音视频流从 UDP 协议被转换成 RTMP（Real-Time Messaging Protocol）协议的过程。
	<br>- 连麦：直播场景里，频道内的观众可以切换角色为主播，与频道内现有主播进行互动，称之为连麦。
	<br>- H264：H.264，又称为 MPEG-4 第 10 部分，高级视频编码（Advanced Video Coding，简称 AVC）。是一种面向块，基于运动补偿的视频编码标准。
	<br>- H265：H.265，又称为 MPEG-H 第 2 部分，高效率视频编码（High Efficiency Video Coding，简称 HEVC）。是一种视频压缩标准。</div>

#### 转码并发频道数

选择**转码并发频道数**可查看转码并发频道数用量。其中：

- 按每日查询时，用量数据为该公司当天每个项目用量峰值之和。
- 按每月查询时，用量数据为该公司当月内每个项目用量峰值之和，而不是每日用量数据之和。

![](https://web-cdn.agora.io/docs-files/1569572103415)

- 单位：个。
- 图表内容：默认显示 H264 总和。若需查看 H264 详细分类或 H265 相关图表，可在图表横坐标下方图例中选择。
  - H264 总和：使用 H.264 编码的视频连麦转码并发数总和。
  - H264 音频：主播的音频总时长。
  - 总音频连麦：使用 H.264 和 H.265 编码的音频连麦转码并发数总和。
  - H264 单主播：使用 H.264 编码的单主播视频转码并发数。

#### 实时消息

选择**实时消息**可查看实时消息用量。其中：

- 按每日查询时，用量数据为该公司当天每个项目用量峰值之和。
- 按每月查询时，用量数据为该公司当月内每个项目用量峰值之和，而不是每日用量数据之和。

![](https://web-cdn.agora.io/docs-files/1569572114027)

- 单位：个。
- 图表内容：日活跃用户数。

### 下载用量

1. 登录[代理商控制台](https://reseller.agora.io/)，在左侧导航栏中点击**用量>下载**。

2. 选择按每日或每月下载，及下载时段。

3. 勾选需要的公司、项目、产品数据，并点击**下载**。你可以在下载的 Excel 文件（默认文件名为 usage）中查看用量数据。

<div class="alert note">为避免数据量过大，下载的数据中不包含没有用量的公司。</div>
